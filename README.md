# Currency Converter
All Currency Converter WebApp built with JavaScript.
This is a simple currency converter application built using JavaScript. It allows users to convert between different currencies based on the latest exchange rates.

Features -
Convert between multiple currencies
Real-time exchange rate updates
User-friendly interface

Usage -
Select the currency you want to convert from in the left dropdown menu.
Select the currency you want to convert to in the right dropdown menu.
Enter the amount you want to convert.
Press the "Convert" button to see the converted amount.

Dependencies -
This project uses the Exchange Rates API to fetch the latest exchange rates.
